Challenge 4: Decide whether to buy Bitcoin

Earlier we learned the instructions to get the price of Bitcoin. Now we want to create a reusable instruction for our friend that will decide whether we want to buy Bitcoin based on the current price. Our instructions are as follows:

  1. Get the current price o Bitcoin in U.S. Dollars
  2. Decide to buy if the cost is below a given price
  3. Tell me whether the decision is to buy or wait
Instruction Library
String get_data_from_website(address)
address is a website, for example http://www.example.com 
String get_price_in_usd(data)
data is a JSON string with the format { "USD" : {"15m" : 16766.22, ... }} 
void tell_me(value)
displays value on the screen
