Challenge 6: Quiz game

Create a quiz where the player is asked three questions. You will come up with the questions, possible answers, and the correct answer for each question. You can create instances of the Question class for each one.>br/> Create a loop that goes through each Question and asks the user for the answer, giving them feedback if they were correct or not.

  1. Create an array of your Questions
  2. Loop through the array of Questions
  3. Loop instruction 1): Ask the user the question
  4. Loop instruction 2): Determine if the answer is correct
  5. Loop instruction 3): Keep track of results
  6. Display the final results as "Number correct: #{number}"
