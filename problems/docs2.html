Challenge 2: Display a personal greeting

We need to ask the user for their name, and then display a personal greeting on the screen.
Use the ask_me instruction to get the user's name. Assign the response to a variable.
Use the create_personal_greeting instruction to create a string that says Hello #{name}.
Once again, use the tell_me instruction to display the greeting string on the screen.

  1. Ask the user for their name and remember the answer
  2. Create the personal greeting string using their name
  3. Display the personal greeting on the screen
Instruction Library
String ask_me(prompt)
Display the given prompt on the screen, then wait for the user to type a value and hit return 
String create_personal_greeting(name)
Creates a string which says Hello #{name} 
void tell_me(value)
displays the value on the screen
