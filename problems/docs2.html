Challenge #2: Display the number of seconds in a day

We need to calculate the number of seconds in a day and then display it on the screen. Calculator instructions in Ruby take the form:
result = value operator value
For example:
secs_in_hour = 60 * 60

  1. Calculate seconds in an hour and remember the result
  2. Multiple that value times 24 to get seconds in a day
  3. Display the final result
Instruction Library
void tell_me(value)
displays the value on the screen
