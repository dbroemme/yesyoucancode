Challenge 3: Days until your Birthday

We want to see how many days are left until our birthday.
Use the ask_me instruction to get the user's birthday. We will use the date format MM/DD. For example, July 4th would be 07/04.
Use the calculate_days_until instruction to calculate how many days are left.
Use the tell_me instruction to display "There are #{number} days left."

  1. Ask the user for their birthday and remember the answer
  2. Calculate how many days are left until that day of the year
  3. Display how many days are left on the screen
Instruction Library
String ask_me(prompt)
Display the given prompt on the screen, then wait for the user to type a value and hit return 
int calculate_days_until(date_string)
Given a date string formatted as MM/DD, returns the number of days left until that date 
void tell_me(value)
displays the value on the screen
