Challenge 3: Display a personal greeting

We need to ask the user for their name, and then tell them "Hello #{name}" on the screen.

Use the ask_me function to get the user's name. Assign the response to a variable.

You can create a dynamic String in Ruby using the following format. Note that the variable name is enclosed within this notation: #{___}
"static text and a #{variable_name}"

  1. Ask the user for their name and remember the answer
  2. Display "Hello #{name}" to the screen
Instruction Library
String ask_me(prompt)
Display the prompt on the screen, then wait for the user to type a value and hit return 
void tell_me(value)
displays the value on the screen
