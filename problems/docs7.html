Challenge 7: Calculator

Create a calculator class that can keep a running total, perform addition, substraction, multiplication, division, and display the current total.
Compute and display the answer to 5 multiplied by 78, and then divide that result by 3.

  1. Define a calculator class
  2. Write a method for each arithmetic operation
  3. Write a method to display the current total
  4. Compute 5 multiplied by 78
  5. Compute that result divided by 3
  6. Display the result
