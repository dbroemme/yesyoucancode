Challenge 5: Countdown Timer

In this challenge, we will run a timer that starts at 10 and goes down to zero. At each step, we will display a message "#{seconds} seconds left" We wait one second in between each step, for a total of 10 seconds. At the end, we display "Done"

  1. Create a loop that will execute instructions 10 times
  2. Loop instruction 1): Print the current count
  3. Loop instruction 2): Wait for one second
Instruction Library
void sleep(seconds)
Waits for the given amount of seconds before going on to the next instruction 
void tell_me(value)
displays value on the screen
